# LGCELL2 ドキュメント

LGCELL2 は、論理回路をグリッドベースの有向グラフとしてモデリングし、ステップ単位でシミュレーションを実行する回路エディタ／シミュレータです。

## 基本コンセプト

- **セル（Cell）**: グリッド座標 `(x, y)` 上に配置される計算単位。`bool` 値（0 / 1）を保持します。
- **ワイヤ（Wire）**: セル間を結ぶ有向辺。**Positive**（そのまま伝搬）と **Negative**（反転伝搬）の 2 種類の極性があります。
- **回路（Circuit）**: セルとワイヤの集合で構成される有向グラフ。

## 回路の動作原理

### セル値の合成

1 つのセルに複数の入力ワイヤが接続される場合、全伝搬値の **OR（最大値）** を取ります。

- 入力ワイヤなし → 前 tick の値を保持
- Positive ワイヤ → 入力元の値をそのまま伝搬
- Negative ワイヤ → 入力元の値を反転して伝搬
- 複数ワイヤ → 全伝搬値の OR

### 万能性

Negative ワイヤを 2 本束ねることで NAND ゲートを構成できます。NAND は万能ゲートであるため、あらゆる論理関数の実現が可能です。

### 伝搬遅延

ワイヤの遅延はセル座標の辞書順 `(x, y)` で自動的に決まります。

| 条件 | 遅延 | 説明 |
|------|------|------|
| `dst >= src`（辞書順） | 即時 | 同一 tick 内で伝搬 |
| `dst < src`（辞書順） | 1 tick | 前 tick の値を使用 |

組合せ回路はワイヤを辞書順で前方に配置すれば 1 tick で結果が確定します。逆方向のワイヤには自動的に 1 tick の遅延が入り、順序回路（フリップフロップ等）を構成できます。
