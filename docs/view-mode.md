# ビューモード（`--view`）

回路シミュレーションの状態を端末にリアルタイム表示する TUI モードです。

## 起動方法

```bash
# ファイルを指定して起動
lgcell2 --view circuit.json
lgcell2 -v circuit.json

# 標準入力から読み込み
cat circuit.json | lgcell2 --view
```

`--view` と `--ticks` は同時に使用できません。両方を指定するとエラーになります。

## 画面構成

```
..........##.._.._..
..........##.._.._..
.._##_....##.._.._..
.._##_..............
....................
tick:42 | running | (0,0)-(19,4)          (q:quit space:pause arrows:scroll)
```

画面は 2 つの領域で構成されます。

- **グリッド領域**（最下行以外）: 各セルの状態をアスキーアートで表示
- **ステータスバー**（最下行）: tick 番号、実行状態、表示範囲、操作ヘルプ

### セルの表示

| 表示 | 意味 |
|------|------|
| `#` | ON（値が 1） |
| `_` | OFF（値が 0） |
| `.` | セルなし |

## キー操作

| キー | 動作 |
|------|------|
| `q` | ビューモードを終了 |
| `Space` | 一時停止 / 再開 |
| `↑` `↓` `←` `→` | 表示領域をスクロール |

## シミュレーション進行

- 0.2 秒間隔で自動的に 1 tick ずつ進行します
- 起動直後は tick 0（初期状態）が表示されます
- 一時停止中はシミュレーションが停止し、画面は最後の状態を維持します

## ステータスバーの読み方

```
tick:42 | running | (0,0)-(19,4)          (q:quit space:pause arrows:scroll)
```

| 項目 | 説明 |
|------|------|
| `tick:42` | 現在の tick 番号 |
| `running` / `paused` | 実行中 / 一時停止中 |
| `(0,0)-(19,4)` | 表示領域の左上座標と右下座標 |

## スクロール

矢印キーで表示領域（ビューポート）を移動できます。初期位置は回路のバウンディングボックスの左上角です。回路の範囲外にもスクロール可能で、範囲外は `.` で表示されます。
